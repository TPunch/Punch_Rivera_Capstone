Procedure

In order to complete this project, the following steps were taken:

1) Create a SQL database which will contain the sensor values from the RPiHat.

2) Create a perl script that will obtain the sensor values from the RPiHat and insert the sensor values and the date and time of the reading into the database.

3) Setup a crontab job which runs the perl script every minute.

4) Setup a web server (apache2) in order to use ChartJS to create charts for each sensor's values over time.

5) Create a php script to access the database and convert the recorded data into JSON format.

6) Create a javascript script in ChartJS that graphs the JSON formatted data for the last 24 hours worth of data.

7) Once the webpage(s) are displaying what is required, setup port forwarding on the network's router so the webpage(s) are accessible from the public internet.

8) Have fun staring at charts!
